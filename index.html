broadway by flatiron

Lightweight application extensibility and composition with a twist of feature reflection.

Installation

Server-side

npm install broadway

Browser-side

Download one of these browser builds (you can build one yourself by executing bin/build in Broadway source tree). Put it anywhere you want and just source it in your HTML (you can also hotlink it from this page if you feel like).

  • broadway@0.1.4 (minified version)
See example of Broadway use in the browser.
